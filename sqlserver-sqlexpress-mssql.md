# MS SQL DB

## Expose SQLEXPRESS using tcp

- Run C:\Windows\SysWOW64\SQLServerManager($version).msc
- Enable TCP/IP properties and set properties/IP Addresses/IPAll/port to 1433
- Restart the SQL Server (SQLEXPRESS) service
